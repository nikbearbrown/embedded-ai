Figure 5.4 — Three ways to lay out activation memory

Each strategy applies to a different operator class; a real engine combines all three.

1 · in-place
ReLU, BN, residual-add
cost: 0 extra buffer
2 · buffer reuse
feedforward layers, ping-pong
cost: 2× largest tensor
3 · scratch pad
im2col tiles, GEMM workspace
cost: arena + tile budget

In-place is free when shapes match; reuse halves arena; scratch is the silent killer when kernels assume desktop matmul.
